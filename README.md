# Git Tutorial

## Initialize a repository
* ``git init``
* ``git status``

## Add files and commit
* ``git add file1 file2 file3``
* ``git commit -m"commit message"``
* ``git log``

## Add more files
* ``git add -A``

## Branch
* ``git branch new-branch``
* ``git checkout new-branch``
* ``git branch -b new-branch``

## Push and Pull
* git clone origin
* git pull
* git push -u origin master
